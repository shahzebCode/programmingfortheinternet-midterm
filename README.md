# Programming for the Internet - Midterm Project

## E-Commerce Microservices

This project implements a containerized microservices backend using ASP.NET Core and Docker.

## Services

- CustomerService
- ProductService
- OrderService
- PaymentService

## How to Run

Clone the repository:

git clone https://github.com/shahzebCode/programmingfortheinternet-midterm

Navigate to the project:

cd programmingfortheinternet-midterm

Run Docker Compose:

docker compose up --build

Services will be available at:

CustomerService → http://localhost:5001  
OrderService → http://localhost:5002  
ProductService → http://localhost:5003  
PaymentService → http://localhost:5004
